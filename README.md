# Blue Coast Software

An independent guild of tools and games, built in the spirit of craft rather than commerce.

## Local Development

1. Open `index.html` in a browser
2. Edit files in `css/` and `js/` as needed
3. No build process required

## Deployment

This is a static site. Deploy to:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Customization

Update these placeholders in `index.html`:
- `YOUR_GITHUB_USERNAME` (appears 2 times)
- `hello@bluecoastsoftware.com` (your email)
- `YOUR_TWITTER` (your Twitter/X handle)
